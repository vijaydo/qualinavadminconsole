Editable table + line chart with Rule of 7 analysis.

== Shortcode ==
[dotank_table_chart]

== Security ==
- No database writes
- No AJAX endpoints
- Output escaped
- Local storage only


== Chart logic ==
- Line 1: Combined (A + B)
- Line 2: Median (Numerator) (overall)


== Legend ==
You can edit the main series legend label directly in the UI. The Median line stays labeled "Median".


== Insights ==
Adds decision-focused Insights section with headline, explanation, evidence, and suggested action.


== Import / Paste ==
Supports CSV import and paste-from-Excel (tab/newline) to quickly populate 100+ rows. Data is saved locally in the browser.
